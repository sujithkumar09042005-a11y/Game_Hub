# Project Memory (Simon Game)

A minimalistic, responsive Memory Game (Simon) built with HTML, CSS, and jQuery. Test your memory by following the pattern of lights and sounds!

## Features

-   **Three Difficulty Levels**:
    -   **Easy**: Standard speed (1s delay).
    -   **Medium**: Faster pace (0.6s delay).
    -   **Hard**: Rapid fire (0.3s delay).
-   **Minimalistic UI**: Clean dark theme with modern typography and animations.
-   **Responsive Design**: optimized for desktop and mobile play.
-   **Sound Effects**: Custom sounds for each color and Game Over.

## How to Play

1.  Open `index.html` in your web browser.
2.  Select a difficulty level: **Easy**, **Medium**, or **Hard**.
3.  Watch the pattern flashes on the colored buttons.
4.  Repeat the pattern by clicking the buttons in the same order.
5.  The pattern gets one step longer each round.
6.  If you miss, the game ends. Select a difficulty to restart.

## Installation

1.  Clone or download this repository.
2.  Ensure you have the `sounds` folder with the required audio files.
3.  Open `index.html` in any modern web browser.
