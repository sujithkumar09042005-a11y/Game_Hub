# Game Hub - Retro Edition

**Game Hub Retro** is a collection of classic browser-based games reimagined with a sleek, synthwave aesthetic. Serving as a central portal, this project features a responsive grid layout, smooth CSS animations, and a custom interactive cursor that enhances the user experience.

The site currently hosts four timeless games:
*   **Memory:** Test your cognitive skills with this card-matching classic.
*   **Rock Paper Scissors:** A quick, battle-ready hand game against the computer.
*   **Snake:** The nostalgic handheld favorite—grow your snake without crashing.
*   **Tetris:** The legendary arcade puzzle game of stacking blocks.

## Key Features

- **Retro Synthwave Theme:** Dark purple/black aesthetics with neon green and pink accents.
- **Interactive UI:** innovative custom cursor, hover effects, and a scrolling grid background.
- **Responsive Design:** Optimized for desktop and tablet play.
- **Pure Web Tech:** Built with semantic HTML5, modern CSS3 variables/animations, and Vanilla JavaScript (No frameworks).

## Included Games

This project serves as a central hub for four distinct web games:

1.  **Memory:** A card-matching game to test your cognitive skills.
2.  **RPS (Rock Paper Scissors):** A classic hand game against the computer.
3.  **Snake:** The retro classic where you grow by eating food while avoiding walls and yourself.
4.  **Tetris:** The legendary tile-matching puzzle game.

## How to Run

1.  Clone the repository or download the source code.
2.  Navigate to the project directory.
3.  Open `index.html` in your modern web browser (Chrome, Firefox, Edge, Safari).
4.  Navigate to any game by clicking its card.

## Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Styling, variables, animations, and Grid/Flexbox layouts.
- **JavaScript (ES6+):** Custom cursor logic and interactive enhancements.
